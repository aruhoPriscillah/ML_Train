# ML Train Project 🚂

## 📌 Project Title

ML Train – Student Activity Prediction Model

---

## 📖 Description (What it does)

This project is a Machine Learning system that predicts whether a student is **Active or Not Active** based on input features.

It uses:

* Data preprocessing and scaling
* A trained model (Random Forest / compressed model)
* Model optimization techniques (pruning, quantisation)

The system can:

* Take student data as input
* Predict activity status
* Output probability scores

---

## ⚙️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Joblib
* (Optional: Streamlit / Flask if you added UI)

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ML_Train.git
cd ML_Train
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate it

**Windows:**

```bash
venv\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

---

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 5. Run the project

Example:

```bash
python main.py
```

OR (if Streamlit):

```bash
streamlit run app.py
```
